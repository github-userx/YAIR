Enabling this option displays modmail separately from your main inbox, with it's own folders.

Select this option to enable compact mode on the inbox. Selecting this option narrows the sidebar, and collapses the card containers.

Use YAIR's dark theme. YAIR supports Reddit Enhancement Suite's night mode, but if you'd prefer to turn on a dark interface seperate from that, or don't use RES, you can do so by selecting this option.

By default, messages in a conversation are sorted from old to new. Showing the first message at the top and the last message at the bottom. This option allows you to reverse the sorting so that messages are sorted from new to old inside a message thread.

By default, new messages are polled for every minute to update notifications in the menu bar. This setting allows for changing that default.

If you switch usernames, this value will not be updated until you view your inbox.

If something has gone wrong (somehow the data in your inbox has been corrupted, messages have gone missing, etc), you can use this option to clear out your inbox and restart from scratch.


Do note however: Reddit normally will not store more than 1000 private messages. YAIR stores messages locally, and thus will store more than 1000 messages. If you have more than a thousand messages, using this option means that you will lose your oldest messages.
